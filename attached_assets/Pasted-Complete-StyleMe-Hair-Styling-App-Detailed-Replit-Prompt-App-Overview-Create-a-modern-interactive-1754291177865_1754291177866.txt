Complete Technical Implementation Prompt
Create a professional hair styling web application with the following specifications:

STRUCTURE:
1. HTML5 semantic structure with proper accessibility
2. Responsive design (mobile-first approach)
3. Single-page application with smooth section transitions

LAYOUT SECTIONS:
1. Header with navigation and auth buttons
2. Hero section with animated background and CTAs
3. Face scanning modal with camera integration
4. Style selection grid with filtering
5. Color palette with swatches
6. Preview section with before/after comparison
7. Sidebar with recently tried, trending, tips, and premium upgrade
8. Footer with links and social media

STYLING REQUIREMENTS:
- Use CSS Grid and Flexbox for layouts
- Implement glassmorphism effects with backdrop-filter
- Add smooth CSS transitions (0.3s ease) for all interactions
- Include hover effects with transform and shadow changes
- Use CSS custom properties for the color system
- Implement responsive breakpoints: 768px (tablet), 1024px (desktop)

INTERACTIVE FEATURES:
1. Camera access and face detection simulation
2. Hairstyle filtering by category, difficulty, and face shape
3. Color picker with predefined palettes
4. Before/after slider comparison
5. Save/share functionality simulation
6. Search functionality for styles
7. Favorite/unfavorite styles
8. Premium upgrade modal

ANIMATIONS:
- Fade-in animations for page load
- Smooth transitions between sections
- Hover effects on cards and buttons
- Loading spinners and progress bars
- Pulse effects for scanning
- Slide animations for mobile menu

JAVASCRIPT FUNCTIONALITY:
1. Mock camera feed using placeholder
2. Style filtering and search
3. Color application preview
4. Modal controls
5. Image comparison slider
6. Local storage for favorites and history
7. Responsive navigation toggle

COMPONENTS TO INCLUDE:
- Navigation menu with mobile hamburger
- Hero section with animated elements
- Modal dialogs for scanning and premium upgrade
- Grid layouts for style cards
- Color palette selector
- Image comparison slider
- Sidebar with multiple sections
- Footer with organized links

TECHNICAL DETAILS:
- Use modern CSS features (Grid, Flexbox, Custom Properties)
- Implement proper semantic HTML
- Add ARIA labels for accessibility
- Include meta tags for responsive design
- Use CSS animations and transitions
- Implement JavaScript ES6+ features
- Add touch/swipe support for mobile

VISUAL EFFECTS:
- Gradient backgrounds and buttons
- Box shadows with multiple layers
- Border radius for modern card design
- Glassmorphism effects on overlays
- Smooth color transitions
- Scale transforms on hover
- Opacity changes for interactions

The app should feel premium, responsive, and engaging with smooth animations throughout. Focus on user experience and modern design trends while maintaining excellent performance.
